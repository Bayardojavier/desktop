Características Principales

  • Dos Formatos: Fichas detalladas (para documentación) y etiquetas compactas (para identificación).
  • Configuración Personalizable: Control total sobre quón campos mostrar y en quón posiciones.
  • Temas y Paletas: Múltiples opciones de colores y estilos visuales.
  • Vista Previa en Tiempo Real: Visualización inmediata de cambios en el diseño.
  • Búsqueda Avanzada: Localizar materiales por código, nombre, contenedor o tipo.
  • Exportación Múltiple: PDF para impresión profesional y PNG para uso digital.

Flujo de Trabajo

  1. Buscar Material:
    • Seleccionar la bodega principal.
    • Elegir tipo de búsqueda: Material o Contenedor.
    • Ingresar código específico o usar el buscador avanzado.
    • Cargar el material seleccionado.
  2. Configurar Ficha:
    • Seleccionar tema/paleta de colores.
    • Configurar elementos principales (título, encabezado).
    • Definir elementos secundarios (subtítulos, información adicional).
    • Posicionar la foto: izquierda, derecha u ocultar.
    • Seleccionar hasta 12 campos para mostrar en la ficha.
  3. Configurar Etiqueta:
    • Elegir tema diferente para la etiqueta si es necesario.
    • Configurar elementos principales y secundarios (más compactos).
    • Decidir posición de la foto (generalmente oculta o pequeña).
    • Seleccionar hasta 3 campos clave para la etiqueta.
  4. Vista Previa:
    • Observar cambios en tiempo real en el panel de preview.
    • Ajustar configuraciones hasta obtener el diseño deseado.
    • Guardar la configuración para uso futuro.
  5. Exportar:
    • Exportar PDF: Genera ficha completa + múltiples etiquetas por página.
    • Exportar PNG: Crea imagen individual de ficha o etiqueta.

Tipos de Documentos

Ficha Tóncnica

  • Formato: Documento detallado de tamaño carta (792px ancho).
  • Contenido: Hasta 12 campos de información organizados en grid.
  • Elementos: Logo, título, foto grande, información detallada, código de barras opcional.
  • Uso: Documentación tóncnica, archivos, especificaciones completas.

Etiqueta de Identificación

  • Formato: Etiqueta compacta (380px ancho) para impresión en etiquetas adhesivas.
  • Contenido: Hasta 3 campos clave más código y nombre.
  • Elementos: Foto pequeña u oculta, código prominente, información esencial.
  • Uso: Identificación física de contenedores, estanterías, equipos.

Campos Disponibles

La herramienta permite seleccionar quón información mostrar en cada documento:

  • Información Básica: Código, nombre, tipo de uso, tipo de material.
  • Ubicación: Bodega principal, secundaria, contenedor.
  • Especificaciones: Marca, modelo, color, dimensiones, peso, precio.
  • Estado: Cantidad disponible, estado del contenedor.
  • Personalizados: Campos extra definidos en el catálogo.
  • Tóncnicos: Fecha de creación, usuario, notas adicionales.

Temas y Personalización Visual

  • Paletas de Color: Múltiples combinaciones predefinidas.
  • Posicionamiento: Control sobre ubicación de elementos (izquierda, derecha, centro).
  • Fuentes: Estilos tipográficos apropiados para impresión.
  • Logos: Integración de logotipo de la empresa.
  • Códigos de Barras: Generación automática para identificación rápida.

Búsqueda y Selección

Búsqueda Directa

  • Ingresar código específico del material o contenedor.
  • Seleccionar bodega y tipo (Material/Contenedor).
  • Carga inmediata del elemento encontrado.

Buscador Avanzado

  • Búsqueda por texto libre (código, nombre, contenedor, tipo).
  • Lista de resultados con vista previa.
  • Selección interactiva del material deseado.

Exportación y Formatos

PDF para Impresión

  • Genera documento profesional listo para impresión.
  • Incluye ficha completa y múltiples etiquetas por hoja.
  • Optimizado para papel carta y etiquetas adhesivas.
  • Compatible con impresoras estándar y plotters.

PNG para Digital

  • Exportación individual de ficha o etiqueta.
  • Formato de imagen de alta calidad.
  • Ideal para uso digital, correos electrónicos, sistemas de gestión.
  • Selección del tipo y código específico a exportar.

Configuraciones Guardadas

  • Guardar Configuración: Almacenar setups personalizados para reutilización.
  • Reset: Restaurar configuraciones predeterminadas.
  • Persistencia: Las configuraciones se mantienen entre sesiones.

Casos de Uso Comunes

  • Identificación de Equipos: Etiquetas para audiovisual, iluminación, sonido.
  • Documentación Tóncnica: Fichas completas para mantenimiento y reparaciones.
  • Inventario Físico: Etiquetas para verificación y conteo de materiales.
  • Organización de Bodega: Identificación de contenedores y estanterías.
  • Control de Calidad: Fichas con especificaciones tóncnicas detalladas.
  • Transferencias: Documentación para movimientos entre bodegas.

Consideraciones Tóncnicas

  • Resolución: Diseños optimizados para impresión de alta calidad.
  • Compatibilidad: Funciona con html2pdf.js para exportación PDF.
  • Almacenamiento: Configuraciones guardadas en el navegador local.
  • Actualización: Cambios en el catálogo se reflejan automáticamente.
  • Validación: Verificación de existencia del material antes de generar documentos.

Limitaciones

  • Fichas limitadas a 12 campos para mantener legibilidad.
  • Etiquetas limitadas a 3 campos para formato compacto.
  • Dependiente de conexión a Supabase para datos actualizados.
  • Formatos de exportación fijos (PDF carta, PNG alta resolución).
