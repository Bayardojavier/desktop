Agregar Material de Hierro

Esta interfaz permite dar de alta nuevos materiales de hierro en el inventario de la bodega. Está especializada para el manejo de estructuras metálicas, perfiles, tubos, piezas y equipos de hierro utilizados en montajes y construcciones.

Características Principales

  • Bodega Principal Fija: La bodega principal está predefinida como "Hierros" para todos los materiales de este tipo.
  • Tipos de Material: Dos categorías principales: Equipo/Estructura y Piezas (accesorios).
  • Dimensiones Específicas: Campos dedicados para Ancho, Largo y Grosor de los materiales.
  • Unidades de Medida: Pulgadas, Centímetros, o Sin medida específica.
  • Peso y Precio: Control preciso del peso en kilogramos y precio unitario.
  • Largo de Piezas: Para accesorios, se puede especificar el largo (ej. 10M, 15M, 150CM).

Flujo de Trabajo

  1. Selección de Ubicación:
    • La bodega principal es fija: "Hierros".
    • Seleccionar o crear una bodega secundaria (ej. "Estructuras", "Piezas", "Tubos").
    • Seleccionar o crear un contenedor específico para el tipo de hierro (Lote, Burra, Andamio, Cuarto, Patio, Balde, Otro).
  2. Configuración del Tipo:
    • Elegir entre "Equipo/Estructura" o "Piezas" (accesorios).
    • Si es "Piezas", aparecerá el campo adicional para especificar el largo.
    • Especificar la cantidad inicial del material.
    • Marcar si el contenedor debe cerrarse despuóns de agregar este material.
  3. Información del Material:
    • Ingresar el nombre del producto (ej. "Perfil C", "Tubo redondo").
    • Especificar una abreviación para el código (opcional, se genera automáticamente).
    • Subir una foto del material (opcional).
    • Seleccionar el tipo de uso (ej. "Montaje", "Estructura").
    • Seleccionar el tipo de material (ej. "Acero", "Aluminio").
  4. Especificaciones Tóncnicas:
    • Elegir la unidad de medida (Pulgadas, Centímetros, Sin medida).
    • Ingresar las dimensiones: Ancho, Largo, Grosor.
    • Especificar el peso en kilogramos.
    • Ingresar el precio unitario.
    • Marcar si es un producto fabricado localmente.
    • Agregar marca y modelo si aplica.
  5. Campos Personalizados:
    • Agregar campos extra según necesidades específicas (Texto, Número, Área de texto, Fecha).
  6. Guardado:
    • Si "Activo" está marcado, se crea el registro en catálogo y un movimiento de ingreso.
    • Si "Activo" está desmarcado, solo se crea el registro en catálogo.

Tipos de Contenedor

Los materiales de hierro se organizan en contenedores específicos según su naturaleza:

  • Lote: Agrupación de materiales similares por características o procedencia.
  • Burra: Contenedor grande para materiales pesados o voluminosos.
  • Andamio: Estructuras temporales de hierro para construcción.
  • Cuarto: Espacio cerrado para almacenamiento de materiales específicos.
  • Patio: Área abierta para materiales que requieren espacio amplio.
  • Balde: Contenedor pequeño para piezas o accesorios.
  • Otro: Categoría genónrica para casos especiales.

Generación de Códigos

El sistema genera códigos únicos siguiendo la jerarquía: Bodega Principal > Secundaria > Contenedor > Material.

Ejemplo de código: HIERROS-ESTRUCTURAS-LOTE01-PRFC-001

Diferencias con Otros Módulos

  • Bodega Principal Fija: Siempre "Hierros", no se puede cambiar.
  • Dimensiones Específicas: Campos dedicados para Ancho x Largo x Grosor.
  • Peso Obligatorio: Campo requerido para cálculos de carga y transporte.
  • Largo de Piezas: Campo adicional específico para accesorios con medidas lineales.
  • Contenedores Especializados: Tipos de contenedor adaptados a materiales de construcción.

Puntos Importantes

  • Las dimensiones y el peso son críticos para el cálculo de estructuras y capacidades de carga.
  • La correcta clasificación por tipo de uso facilita la búsqueda para proyectos específicos.
  • Los materiales de hierro requieren un control especial debido a su valor y peso.
  • El precio unitario es esencial para presupuestos de construcción y montajes.
  • Los campos personalizados permiten registrar propiedades específicas como resistencia, tratamiento, etc.
