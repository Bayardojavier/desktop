Características Principales

  • Edición de Propiedades: Modificar nombre, tipo y estado del contenedor.
  • Gestión de Estado: Abrir o cerrar contenedores para controlar altas de nuevos materiales.
  • Movimiento de Módulos: Reubicar módulos entre diferentes contenedores.
  • Extracción de Módulos: Sacar módulos de contenedores sin eliminarlos.
  • Histórico Intacto: Los cambios no afectan movimientos históricos del inventario.

Flujo de Trabajo

  1. Selección de Ubicación:
    • Seleccionar una bodega secundaria de la lista disponible.
    • Seleccionar el contenedor específico que se desea modificar.
  2. Edición del Contenedor:
    • Modificar el nombre visible del contenedor.
    • Cambiar el tipo del contenedor (ej. Case, Rack).
    • Marcar/desmarcar como "Contenedor cerrado" para bloquear nuevas altas.
    • Guardar los cambios realizados.
  3. Gestión de Módulos:
    • Revisar la lista de módulos contenidos en el contenedor seleccionado.
    • Seleccionar uno o más módulos para mover o extraer.
    • Elegir el contenedor destino para mover los módulos.
    • Utilizar "Mover a..." para reubicar módulos entre contenedores.
    • Utilizar "Sacar del contenedor" para extraer módulos sin asignarles un nuevo contenedor.

Estados del Contenedor

  • Abierto: Permite agregar nuevos materiales al contenedor.
  • Cerrado: Bloquea la posibilidad de agregar nuevos materiales, pero permite consultar y mover los existentes.

Operaciones Disponibles

Editar Contenedor

  • Cambiar Nombre: Actualiza el nombre visible del contenedor.
  • Cambiar Tipo: Modifica la clasificación del contenedor.
  • Cambiar Estado: Abrir/cerrar el contenedor para controlar altas.

Gestionar Módulos

  • Mover entre Contenedores: Reubica módulos de un contenedor a otro dentro de la misma secundaria.
  • Extraer Módulos: Remueve módulos del contenedor sin asignarles una nueva ubicación.

Consideraciones Importantes

  • Histórico Intacto: Ninguna operación afecta los movimientos históricos del inventario.
  • Actualización de Materiales: Al cambiar el nombre del contenedor, se recomienda actualizar tambiónn los códigos de los materiales contenidos.
  • Contenedores Cerrados: Los contenedores cerrados aparecen marcados en la lista con el indicador "[CERRADO]".
  • Selección Múltiple: Se pueden seleccionar múltiples módulos para operaciones masivas.
  • Validación de Destino: Al mover módulos, el sistema valida que el contenedor destino exista y estón abierto.

Casos de Uso Comunes

  • Reorganización: Mover módulos entre contenedores para optimizar el almacenamiento.
  • Mantenimiento: Cerrar contenedores completos para evitar modificaciones accidentales.
  • Corrección: Actualizar nombres o tipos de contenedores por errores de digitación.
  • Separación: Extraer módulos específicos para reubicarlos en contenedores dedicados.

Limitaciones

  • No se pueden eliminar contenedores con módulos activos.
  • No se pueden mover módulos entre diferentes bodegas secundarias.
  • Los cambios solo afectan al catálogo, no a movimientos históricos.
