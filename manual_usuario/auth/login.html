Inicio de Sesión

La pantalla de inicio de sesión es el primer paso para acceder al sistema Absolute. Aquí se valida la identidad del usuario mediante credenciales únicas, asegurando un acceso seguro y controlado a las funcionalidades del sistema.

Descripción General

Esta pantalla presenta un formulario simple y seguro donde los usuarios registrados pueden ingresar sus credenciales. El sistema valida la información contra la base de datos y, en caso de éxito, redirige al usuario al dashboard correspondiente según su rol asignado.

Seguridad

El proceso de autenticación utiliza encriptación de contraseñas y validación en tiempo real. Las sesiones se manejan de forma segura con tokens temporales.

Pasos para Iniciar Sesión

  • 1

    Acceder a la Pantalla

    Abre la aplicación Absolute y navega a la pantalla de login. Si ya tienes una sesión activa, serás redirigido automáticamente.

  • 2

    Ingresar Usuario

    Escribe tu nombre de usuario en el campo correspondiente. Este debe ser único y fue asignado durante el registro.

  • 3

    Ingresar Contraseña

    Ingresa tu contraseña de forma segura. El campo oculta los caracteres para mayor privacidad.

  • 4

    Confirmar Acceso

    Haz clic en "Ingresar" para validar las credenciales. Si son correctas, accederás al sistema según tu rol.

Preguntas Frecuentes

¿Qué hacer si olvido mi contraseña?

Contacta al administrador del sistema para restablecer tu contraseña. Actualmente no hay sistema de recuperación automática.

¿Puedo tener múltiples sesiones?

Sí, el sistema permite múltiples sesiones activas desde diferentes dispositivos, pero se recomienda cerrar sesiones inactivas por seguridad.

¿Qué roles existen?

Los roles incluyen: Administrador, Bodega, Logística, Contabilidad, RRHH y Ventas, cada uno con permisos específicos.
