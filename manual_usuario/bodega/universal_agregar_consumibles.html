Agregar Material Consumible

Esta interfaz permite dar de alta nuevos materiales consumibles en el inventario de la bodega. Los consumibles son materiales que se utilizan y desgastan durante el proceso productivo, como cintas adhesivas, brochas, repuestos, pinturas, adhesivos, y otros artículos similares.

Características Principales

  • Modo Automático: El sistema opera en modo automático, creando un contenedor si no hay ninguno seleccionado, o agregando material al contenedor existente.
  • Gestión de Cantidad: Permite especificar la cantidad inicial del material consumible que se está agregando al inventario.
  • Unidades de Medida: Soporta diferentes unidades como Unidad, Par, Caja, o Sin medida específica.
  • Cantidad por Unidad: Permite especificar la cantidad contenida en cada unidad (ej. una caja contiene 50 unidades).
  • Producto Fabricado Local: Opción para marcar si el producto fue fabricado localmente.
  • Cierre de Contenedor: Opción para bloquear un contenedor y evitar que se agreguen más materiales a ónl.

Flujo de Trabajo

  1. Selección de Ubicación:
    • Seleccionar una bodega secundaria existente o crear una nueva.
    • Seleccionar o crear un contenedor dentro de la secundaria (Caja, Estante, Gabeta, Rack, etc.).
  2. Configuración del Material:
    • Ingresar el nombre del producto consumible.
    • Especificar una abreviación para el código (opcional, se genera automáticamente si se deja vacío).
    • Seleccionar el tipo de uso y tipo de material.
    • Elegir la unidad de medida apropiada.
    • Especificar la cantidad por unidad y el precio unitario.
  3. Información Adicional:
    • Subir una foto del material (opcional).
    • Seleccionar un color representativo.
    • Ingresar marca y modelo si aplica.
    • Agregar campos personalizados si es necesario.
  4. Guardado:
    • Si "Activo" está marcado, se crea tanto el registro en el catálogo como un movimiento de ingreso en el inventario.
    • Si "Activo" está desmarcado, solo se crea el registro en el catálogo sin afectar el stock.
    • Si se marca "Cerrar contenedor", se bloquea para evitar agregar más materiales.

Generación de Códigos

El sistema genera códigos únicos para los materiales consumibles siguiendo una estructura jerárquica similar a otros módulos, heredando el código del contenedor y añadiendo componentes específicos del material.

Ejemplo de código: 00123-CONSUM-CAJA-01-ADHES-CINTA

Diferencias con Otros Módulos

  • Unidades de Medida: Más flexibles que audiovisual (Unidad, Par, Caja, Sin medida).
  • Cantidad por Unidad: Permite especificar subdivisiones (ej. una caja contiene X unidades).
  • Producto Fabricado Local: Campo específico para consumibles producidos internamente.
  • Tipos de Contenedor: Adaptados para consumibles (Caja, Estante, Gabeta, Rack).

Puntos Importantes

  • Los consumibles son materiales que se agotan con el uso, por lo que el control de inventario es crítico.
  • La correcta clasificación por tipo de uso y material facilita la búsqueda y el reporte.
  • El precio unitario es importante para el cálculo de costos en producciones.
  • Los campos personalizados permiten adaptar la información a necesidades específicas de cada tipo de consumible.
