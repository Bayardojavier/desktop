Agregar Material Audiovisual

Esta interfaz es una de las más completas y fundamentales del sistema, ya que permite dar de alta nuevos materiales en la bodega de Audiovisual. El formulario está diseñado para capturar todos los detalles necesarios para una gestión de inventario precisa y organizada, siguiendo una estructura jerárquica de Bodega Principal > Bodega Secundaria > Contenedor > Material.

Flujo de Trabajo

  1. Selección de Ubicación:
    • Bodega Principal: Es fija y corresponde a "Audiovisual".
    • Bodega Secundaria: Se debe seleccionar o crear una bodega secundaria (ej. "Cables", "Audio", "Luces").
    • Contenedor: Dentro de la secundaria, se selecciona un contenedor existente (ej. "Case 01", "Rack 05"). Si el contenedor no existe, se debe crear primero.
  2. Modo de Alta: El formulario permite dos modos principales:
    • Agregar Contenedor: Permite registrar un nuevo contenedor (Rack, Case, Caja, etc.) dentro de una bodega secundaria.
    • Agregar Material: Permite registrar un nuevo material dentro de un contenedor ya seleccionado.
  3. Identificación del Material:
    • Tipo de Item: Se define si es un "Equipo electrónico" (con número de serie único) o un "Accesorio" (gestionado por cantidad).
    • Nombre y Abreviación: Se asigna un nombre descriptivo y una abreviación para la generación del código.
    • Código: El sistema genera un código único y estructurado basado en la ubicación, tipo y otros atributos del material. Este código es fundamental para el seguimiento.
  4. Detalles y Especificaciones:
    • Foto: Se puede subir una imagen del material.
    • Tipos: Se puede especificar el tipo de uso (Producción, Concierto) y el tipo de material (Audio, Video, Iluminación).
    • Dimensiones y Peso: Campos para registrar las características físicas del material.
    • Precio y Marca/Modelo: Información de compra y del fabricante.
    • Campos Extra: Se pueden añadir campos personalizados para capturar información específica no contemplada en el formulario estándar.
  5. Cantidad y Guardado:
    • Cantidad: Si es un "Equipo", se crearán N registros individuales con códigos consecutivos. Si es un "Accesorio", se crea un solo registro con la cantidad especificada.
    • Cerrar Contenedor: Opción para bloquear un contenedor y no permitir que se agreguen más materiales a ónl.
    • Guardar: Al guardar, se crea el registro en el catálogo de `catalogo_audiovisual` y, si la opción "Activo" está marcada, se genera un movimiento de ingreso en `movimientos_bodega_audiovisual`.

Generación de Códigos

El sistema utiliza un mecanismo configurable para generar los códigos de los materiales. El código hereda la estructura de su contenedor y se le añaden partes configurables (como el tipo de material y una abreviación). Esto asegura que cada ítem tenga un identificador único y fácil de interpretar.

Ejemplo de código de material: 00123-AV-AUDIO-CON-MIC-SHURE-SM58-01

  • 00123: Consecutivo único.
  • AV-AUDIO: Bodega Principal y Secundaria.
  • CON-MIC: Contenedor.
  • SHURE-SM58: Partes configurables (ej. marca, modelo).
  • 01: Índice del hijo (si es un equipo).

Puntos Clave

  • La correcta clasificación y llenado de este formulario es crucial para la integridad del inventario.
  • El modo "Ficticio" (desmarcando "Activo") permite registrar un material en el catálogo sin que afecte el stock real, útil para planificación o plantillas.
  • La gestión de fotos se realiza a travóns de Supabase Storage, centralizando los recursos visuales.
