Explicación Paso a Paso

PASO 1: VENTAS - Creación del Evento

  1. Crear Cliente: Registrar datos del cliente en el sistema
  2. Crear Servicio: Definir el servicio/producto a ofrecer
  3. Crear Cotización: Generar presupuesto basado en cliente y servicio
  4. Crear Evento: Convertir cotización aprobada en evento activo

Una vez creado el evento, se pasa el control a Logística

PASO 2: LOGÍSTICA - Solicitud de Despacho

Requisitos previos:

  • Base de persona de RRHH (para elegir encargado)
  • Receta creada en el sistema
  • Evento creado por Ventas

Tipos de solicitud:

  • Despacho de audiovisual (material audiovisual obligatorio)
  • Solicitud de materiales (materiales consumibles)

La solicitud se envía automáticamente a Bodega

PASO 3A: BODEGA - Registro Inicial (PROCESO PARALELO)

Este paso debe hacerse ANTES de que Logística pueda crear solicitudes:

  1. Usar códigos "agregar" en el sistema
  2. Ingresar material al catálogo
  3. Crear movimiento inicial de inventario

Sin este paso, no hay materiales disponibles para solicitar

PASO 3B: BODEGA - Procesamiento de Solicitud

  1. Recibir solicitud de Logística
  2. Validar stock disponible
  3. Crear el DESPACHO correspondiente
  4. Registrar movimiento de salida

Ahora el material está "despachado" y se puede devolver

PASO 4: DEVOLUCIÓN

REGLA IMPORTANTE: No se puede hacer devolución sin un despacho previo

Tipos de devolución:

  • Devolución por Despacho: Devolver materiales de un despacho específico
  • Devolución por Evento: Devolver materiales de todos los despachos de un evento
  1. Seleccionar despacho/evento
  2. Seleccionar materiales a devolver
  3. Registrar cantidades devueltas
  4. Actualizar inventario

Leyenda del Flujo

Ventas: Punto de inicio para cualquier transacción comercial
Clientes: Base fundamental requerida por todos los módulos
Servicios: Catálogo necesario para crear cotizaciones
Cotizaciones: Presupuestos que generan eventos de venta
Eventos: Transacciones activas que requieren logística
Logística: Coordina el transporte y genera solicitudes
Solicitudes: Pedidos que activan el proceso de despacho
Despacho: Proceso final que requiere inventario de bodega
Bodega: Control de stock y materiales disponibles

Guía de Dependencias

Para crear un DESPACHO necesitas:

  • Solicitud de Logística
  • Evento de Venta activo
  • Cotización aprobada
  • Servicio seleccionado
  • Cliente registrado
  • Inventario disponible en Bodega

Orden Recomendado:

  1. 1. Registrar Cliente
    Base de datos de clientes
  2. 2. Configurar Servicios
    Catálogo de productos/servicios
  3. 3. Crear Cotización
    Presupuesto para el cliente
  4. 4. Generar Evento de Venta
    Transacción activa
  5. 5. Solicitar Logística
    Coordinar transporte
  6. 6. Procesar Despacho
    Entrega final con control de bodega
